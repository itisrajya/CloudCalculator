# 🌥️ Cloud Calculator – Blazor + .NET 8/9

A cloud-powered, research-friendly mathematical computation tool built with **ASP.NET Core**, **Blazor**, and **parallel computing**.

Designed for:
- PhD scholars  
- Researchers  
- Engineering students  
- ML/AI learners  

Compute heavy mathematical operations directly from your browser.

---

## 🚀 Features

### 🔢 Matrix Operations
- Matrix × Matrix multiplication  
- Vector & matrix parsing  
- Parallel computation for large matrices  

### 📈 Regression Engine
- Linear Regression  
- Shows slope (m), intercept (c) and error  

### 🧭 Optimization Solver
- Gradient Descent algorithm  
- Adjustable learning rate & iterations  
- Computes optimal value for cost functions  

### 🌐 Cloud Web App
- Blazor-based minimal UI  
- Fast, responsive, modern web interface  
- JSON API endpoints available for external integration
